This is an app that is intended to allow it's users to execute the Pomodoro style of studying. To learn more about Pomodoro technique, check out this article. It is currently in it's Minimum Viable Product (MVP) state, meaning that the timer can/will count down based on the times you select in the drop downs. This web applicaiton is not perfect by any means, please feel free to help make it better by contributing on the repository.
